# ScrToQQ
发送电脑截图给 QQ 好友或 QQ 群。  

### 目的
如果你的电脑上尚未安装 QQ，而又想要给好友分享屏幕内容，手机拍照？太不友好了。通过一些方法把截图发送到手机，再到手机上去发送给好友？太麻烦了。  
你可以试试 ScrToQQ。  

### 使用方法
将截图上传至网站后，你可以  
* 点击分享，登录 QQ 号后直接在网页内选择好友或群进行发送，丝毫不影响手机操作；  
* 使用手机 QQ 扫一扫扫描二维码，选择好友或群立即发送，免去登录过程。

分享截图后可以手动删除，也可以等待 7 天后服务器自动清理。
